# AGENTS

Minimal developer map for this repo.

## Quick dev

- `vp install` - install workspace deps
- `vp run dev` - run API + web together
- `vp run test` - run all tests
- `vp run lint` - type/lint checks
- `vp run build` - build shared, API, and web

## Docs (TOC)

- `docs/architecture.md` - system shape + request flow
- `docs/frontend.md` - web app structure + frontend commands
- `docs/backend.md` - API/backend structure + backend commands
- `docs/deployment.md` - deployment steps
- `docs/prd.md` - product requirements
